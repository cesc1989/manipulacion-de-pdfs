# Generación y Manipulación de PDFs en Ruby

Repo para probar varias librerías para generar y manipular PDFs en Ruby y Rails.

Librerías

- WickedPDF ✅
- Prawn ✅
- HexaPDF
- Dhalang ⚠️
- Origami
- Weasyprint
- Grover ✅
- CombinePDF ✅
- Invoice Printer ✅
- Rails PDF
- Receipts ✅

## Getting Started

### Requirements

Install on your computer:

- Ruby 3.1.0
- Bundler gem: `gem install bundler`
- PostgreSQL
- NodeJS(required JavaScript runtime environment)

After clone project, run:

```bash
$ bundle install
$ rails db:create
$ rails db:schema:load
```

### Run Server

Run server with:

```bash
$ foreman start
```

### Tests

To run test suite use `rspec` command:

```bash
$ rspec ./spec
```
